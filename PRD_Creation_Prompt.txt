Create a PRD for a new desktop application that will be used to configure AlertMailService.  This application will target the Windows platform.  It will be used to create, update, delete alert emails defined in the SQL server table used by the MailAlertServer and emailed by the MailRequestServer.  It will also allow the editting of the mail server configuration information.  It will provide a view into the MailRequestServer log that can be refreshed on demand.  It must also have settings that can be configured by the user for making the connection to the SQL Server and Alert Mail Service database.  Once this application is functional, it will be added to the AlertMailServiceInstaller so that it is installed and configured at the same time as the MailAlertServer and MailRequestServer.  Because of this, the database connection settings must be configurable by the AlertMailServiceInstaller application.  Plan for the settings to be saved in a file.  The application shall be a stand-a-lone electron based desktop application.  The alert mails edit must provide a means of test executing the trigger SQL query and message SQL query and previewing the results of these queries.  Simple syntax error checking shall be included to validate the entries and alert the user of an invalid entry.  The interface shall provide a selectable light or dark mode, with the dark mode being the default.  Source code for the MailAlertServer, MailRequestServer and AlertMailService are available for reference in the /home/dave/gitWorkspace folder each under their respective folders.  No changes shall be made to these other applications at this time.
